# Custom PCB Bring-Up

The custom PCB provides the mechanical and electrical attachment between the T-SIM7080G-S3 base and the Grove Vision AI V2 stick-on module. Its source is included as an external submodule:

```text
external/t-sim-motor-shield/
```

Use that repo for PCB source, documentation, and motor shield validation firmware. Keep base application firmware in `firmware/`.
